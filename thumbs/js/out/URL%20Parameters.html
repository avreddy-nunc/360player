URL Parameters

URL Parameters

Source:
  • main.js, line 3
URL Parameters are used to extend the features and functionality of the player and are not mandatory, any parameter can be used only to activate respective feature otherwise defaults will be considered

Members

(static) gyroscope_spin

Toggles the automatic spinning of player using mobile device gyroscope data
Note This feature is completely subjected to the availability of gyroscope api in mobile devices

Source:
  • main.js, line 134
Example
<-- 360 player url -->?gyroscope_spin=1 // 1 enables the feature i.e. Spins the player according to gyroscope data
<-- 360 player url -->?gyroscope_spin=0 // 0 or no param disables the feature i.e. Gyroscope data of device is ignored

(static) hide_buttons

Toggle the display of controls and buttons on player for maximum player visibility. This feature is implemented for emebedding the 360 player in android and ios applications without feature buttons on it

Source:
  • main.js, line 91
Example
<-- 360 player url -->?hide_buttons=1 // 1 for enabling the feature i.e. Hiding the buttons on player
<-- 360 player url -->?hide_buttons=0 // 0 or no parameter disables the feature i.e. Unhides the buttons on player

(static) player_width

Maximum width of the player allowed. Current default value is 1200 and can be modified in player_360_settings.php

Source:
  • main.js, line 84
Example
<-- 360 player url -->?player_width=1200 //specify the maximum needed width of the player

(static) thumbs_out

Toggles the position of feature thumbnails to below the player and on the player.
Note: This feature is ignored without thumbs_player feature enabled and this feature also effects the aspect ratio of the player

Source:
  • main.js, line 126
Example
<-- 360 player url -->?thumbs_player=1&thumbs_out=1 // 1 enables the feature i.e Pushes the thumbnail slider to the below the player.
<-- 360 player url -->?thumbs_player=1&thumbs_out=0 // 0 or no parameter disables the feature i.e. Feature thumbnail slider will be displayed on the player

(static) thumbs_player

Toggle the functionality of Feature thumbnails slider on 360 Player, Allowed values are 0 and 1

Source:
  • main.js, line 118
See:

Example
<-- 360 player url -->?thumbs_player=1 // 1 for enabling the feature i.e. Generating and displaying the thumbnails slider on the player
<-- 360 player url -->?thumbs_player=0 // 0 or no parameter disables the feature i.e. No feathure thumbnails slider is displayed on the player. Default feature button will be displayed
